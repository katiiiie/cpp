# ИИ-Агент для подготовки к дедлайнам

Пользователь предоставляет план обучения по какому-либо своему предмету или учебные материалы по этому предмету и 
предоставляет дополнительную информацию и цели:
- срок обучения,
- предпологаемый процент усвоенного материала на момент конца срока,
- количество времени, которое он готов тратить в день или всего
- процент усвоенного материала на данный момент

ИИ-агент в свою очередь отправляет пользователю возможное распределение обучения по дням на период всего
срока обучения, если это возможно, или предоставляет альтернативный вариант, если выставленные параметры пользователя
кажутся ему невыполнимыми, а также строит план обучения и помогает с организацией учебного процесса.

- Что делает агент: Анализирует учебные цели пользователя, сроки, доступное время и текущий прогресс, чтобы сгенерировать реалистичный и сбалансированный план обучения на каждый день.

- Для какой аудитории: Студенты, школьники и все, кто сталкивается с необходимостью структурированной подготовки к экзаменам или освоения нового учебного материала.


## Установка и сборка
```bash
git clone <ваш-репозиторий>
cd <папка>
mkdir build && cd build
cmake ..
make -j
cd <build>
```

## Для доступа к локальной моедли выполните следующие действия: ( При сборке указывайте параметры вашей системы)
```
git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp
cmake -B build -DCMAKE_BUILD_TYPE=Release -DGGML_METAL=ON -DCMAKE_OSX_ARCHITECTURES=arm64
cmake --build build --config Release -- -j$(sysctl -n hw.ncpu)
```


Скачайте нужную для вас модель и поместите в папку models рядом с папкой склонированного проекта llama.cpp, Затем запустите сервер с моделью.
```
./llama.cpp/build/bin/llama-server \
  -m models/Имя-Вашей-Модели \
  -c 4096 -ngl 999 \
  --host 127.0.0.1 --port 8080
```


## Запуск:
В папке вашего склонированного репозитория откройте файл config.json, в поле api_key вставьте ваш ключ. 
В поле mode укажите предпочитаемый режим (local/online), по умолчанию стоит online. Режим так же можно указать 
при запуске программы (об этом ниже).

Для запуска с режимом из config.json выполните команду: 
```
./ai_agent
```

Для запуска с выбором режима выполните команду: 
```
./ai_agent --local
```
для режима с локальной моделью,
```
./ai_agent --online
```
для режима с онлайн моделью.


Примеры использования: 

1 вариант:


```
Привет! у меня дедлайн через 5 дней, будет экзамен по алгебре. Мне нужно к этому моменту научиться решать задачи на сложение, умножение и деление комплексных чисел. Помоги мне спланировать подготовку
```


2 вариант:


```
- Дедлайны/экзамены: письменный экзамен алгебра через 3 дня; - Задачи: сложение, вычитание, деление и умножение комплексных чисел; - Доступное время: каждый день по 2 часа.; - Прогресс: прочитала про сложение; - Сложности/риски: плохо знаю умножение; - Материалы: учебник по алгебре Мерзляк; - могу заниматься каждый день 2 часа.
```
